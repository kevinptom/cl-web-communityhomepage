# 🌟 Biography Website

A simple and attractive Biography Website built using **HTML** and **CSS**.

## 📖 Project Description

This project is a responsive biography webpage that presents information about a person in a clean and visually appealing format. The website includes styled navigation menus, customized links, and a well-structured layout created with HTML and CSS.

## ✨ Features

- 🎨 Attractive page design
- 🧭 Styled navigation menu
- 🔗 Custom link styling
- 📱 Responsive layout
- 📝 Biography content section
- 💻 Built with pure HTML and CSS

## 🛠️ Technologies Used

- HTML5
- CSS3
- Visual Studio Code
- Git
- GitHub
